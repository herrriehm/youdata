Über Kontakt Datenschutz
Mit der neuen Datenschutzgrundverordnung erhalten EU-Bürger*innen bessere Durchsetzungsmöglichkeiten für den Dateschutz. Mehr Informationen dazu erhält man auf der Seite DeineDatenDeineRechte.de. Von dieser stammt auch das Musterschreiben, welches wir verwenden.

Die ursprünglen Daten für die Unterhmen und Behörden stammen von selbstauskunft.net und stehen unter CC BY-NC-SA 3.0. Du bist herzlichst dazu eingeladen, neue Unternehmen hinzuzufügen bzw. die Daten zu verbessern. Die Daten werden auf Github gespeichert: https://github.com/jfilter/youdata-data.

Auch bei der Software können gerne Leute mithelfen: https://github.com/jfilter/youdata
